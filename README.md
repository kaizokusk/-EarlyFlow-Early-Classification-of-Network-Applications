

# EarlyFlow: Early Classification of Network Applications
COL7560 Course Project

**Project Description:** EarlyFlow is a machine learning-based system designed to classify network traffic into application categories (e.g., streaming, gaming, conferencing) using only the first few packets of a flow. By analyzing early-flow metadata, the system aims to minimize classification latency and bypass the need for full-flow observation or deep packet inspection (DPI) of encrypted payloads.
